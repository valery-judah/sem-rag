# Agentic Docs

## Purpose
Reusable docs framework for feature design and implementation playbooks.

## How to create a new feature doc set
1. Create folder: `docs/features/<feature-name>/`
2. Copy templates from `docs//docs/features/parsers//templates/`
3. Rename template files by removing `.template`
4. Fill `01_rfc.md` first, then align all other artifacts to it

## Required files
- `00_context.md`
- `01_rfc.md`
- `02_user_stories.md`
- `03_design.md`
- `04_workplan.md`

## Optional files
- `05_test_plan.md`
- `06_rollout.md`
